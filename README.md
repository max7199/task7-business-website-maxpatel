# NexaStudio - Business Website

## 📌 Project Overview

NexaStudio is a modern and responsive business website developed using HTML, CSS, and JavaScript. The project showcases a digital agency that provides web development, mobile app development, UI/UX design, cloud solutions, digital marketing, cybersecurity, and AI integration services.

## 🚀 Features

* Responsive Multi-Page Website
* Modern UI/UX Design
* Mobile-Friendly Navigation Menu
* Animated Statistics Counter
* Contact Form with Success Message
* Back-to-Top Button
* Smooth Scrolling Effects
* Interactive Service Sections
* About Company Page
* Contact Information Page
* Professional Business Layout

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
